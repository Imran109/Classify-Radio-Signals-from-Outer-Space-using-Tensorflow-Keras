# Classify-Radio-Signals-from-Outer-Space-using-Tensorflow-Keras
Building and training a convolutional neural network (CNN) in Keras from scratch to classify 2D spectograms of deep space radio signals. 
## Packages Used
```bash
Tensorflow
Sklearn
Matplotlib
Numpy
Pandas
seaborn
```
## Code
